What are some of the ways that you could change your HTML on your Demo app to be more accessible? (Remember, you can use the Section tag with jQuery, for instance)

Using proper formatting in our code. WIth html5 we have been introduced a new way to format our pages in way that makes sense. We have been given a new way to call our divs so that not only it will make sense to us but for users that are using devices or software like screen readers. If by simply placing our articles within the article tag a user using jaws for instance will be able to find all the articles within a document instead of having 100 different divs that jaws just starts reading indefinitely until the user hears what they might be looking for.

Visit a video service on the web (like YouTube) and search for "Screen Reader Demo" and "Mobile Screen Reader" - what are some of the common frustrations (or common support mechanisms) that you see among the examples that are given - how could you fix them (or use them)?

These applications require that the content the user is accessing is written in a manner the app is looking for. Using proper divs and tags within our code can ensure all users can access and enjoy what we are putting online.

Devices: Please list two devices (and some of their features) that are made specifically to support Accessibility.

iOS has accessibility options built right into it. One feature is the flash led and vibrate on notification, some do this already of course but the customization can go even further. Now in iOS6 you can record vibration tones and set them to certain task on the phone. So for someone who can hear we might use a ringtone for a contact but now a deaf user can create a serious of custom vibrations for certain contacts of notifications like sms or email.

Mobile Apps: Please list two apps (either Android or iOS, or both) that are made specifically to support Accessibility.

There is an app called aidColors on iOS. The application allows the user to take a picture of a piece of clothing and it will state the color back to the user. Another iOS app is daisy worm it allows the user to load ebooks and have them read back to them.
